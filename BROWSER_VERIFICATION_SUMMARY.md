## 📊 数据格式验证

### TLE（Two-Line Element）格式

通过实际TLE数据验证了标准格式：

**第0行**: 卫星名称
```
ISS (ZARYA)
```

**第1行**: 基本信息
```
1 25544U 98067A   25330.73812712  .00011313  00000+0  21242-3 0  9996
  │     │  │       │               │          │         │         │
  │     │  │       │               │          │         │         └─ 校验和
  │     │  │       │               │          │         └─ 元素集编号
  │     │  │       │               │          └─ B*拖曳项
  │     │  │       │               └─ 平均运动一阶导数
  │     │  │       └─ 历元时间（年.日）
  │     │  └─ 国际编号
  │     └─ 分类（U=非保密）
  └─ NORAD目录号
```

**第2行**: 轨道参数
```
2 25544  51.6309 221.6882 0003815 171.0984 189.0072 15.49104478540371
  │       │        │        │       │        │        │          │
  │       │        │        │       │        │        │          └─ 圈数
  │       │        │        │       │        │        └─ 平均运动（圈/天）
  │       │        │        │       │        └─ 平近点角
  │       │        │        │       └─ 近地点幅角
  │       │        │        └─ 偏心率
  │       │        └─ 升交点赤经
  │       └─ 倾角
  └─ 卫星编号
```

**结论**: TLE格式完整，包含提案中所需的所有轨道参数。
